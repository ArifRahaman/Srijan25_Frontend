# Srijan'25 Frontend

Srijan'25 is a React-based frontend application designed for the F.E.T.S.U. Presents Srijan event at Jadavpur University. The application provides an interactive interface for users to explore event details, merchandise, and more.

## Features

- **Landing Page**: Welcome screen with animations and navigation options.
- **Event Management**: View event listings and register for events.
- **Merchandise Page**: Explore and order event merchandise with size and color options.
- **Dashboard**: User dashboard accessible through protected routes.
- **Responsive Navbar**: Dynamic navigation bar with mobile support.
- **Footer**: Provides quick links and legal information.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: CSS, Tailwind CSS
- **Routing**: React Router
- **Build Tool**: Vite

## Installation Instructions

1. **Clone the repository**:
   ```
   git clone https://github.com/ArifRahaman/Srijan25_Frontend.git
   cd Srijan25_Frontend
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Start the development server**:
   ```
   npm run dev
   ```

4. **Build for production**:
   ```
   npm run build
   ```

## Usage Guide

- **Home Page**: Access the home page to see event highlights and navigation options.
- **Event Page**: Visit `/events` to see a list of events and register for them.
- **Merchandise**: Go to `/merchandise` to choose and order event-related merchandise.
- **Dashboard**: Log in to access the user dashboard at `/dashboard`.

### Example

To register for an event, navigate to the event page, select an event, and fill in the registration form with your details.
